# 学生请假事务管理系统 — 说明文档

## 项目信息

| 项目 | 详情 |
|------|------|
| 课程 | 数据库课程设计，2025-2026 第 2 学期 |
| 项目名称 | 学生请假事务管理系统 |
| 开发人员 | 汤俊豪 |
| 技术栈 | MySQL 8.0 + Spring Boot + MyBatis + Vue 3 + Axios |

## 系统角色

| 角色 | 功能 |
|------|------|
| **学生** | 提交请假申请、查看自己的请假记录和审批状态 |
| **班主任** | 查看本班待审批请假、审批通过/驳回、查看审批历史和班级统计 |
| **教务管理员** | 管理学生/教师/班级数据、全校请假统计报表 |

## 数据库设计

### 表结构

| 表名 | 说明 | 主键 | 主要字段 |
|------|------|------|----------|
| Class | 班级表 | class_id | class_name, department, grade |
| Student | 学生表 | student_id | name, gender, password, phone, email, class_id(FK) |
| Teacher | 班主任表 | teacher_id | name, gender, password, phone, email, class_id(FK,UNIQUE,NOT NULL) |
| Admin | 教务管理员表 | admin_id | name, password, phone, email |
| LeaveRecord | 请假记录表 | leave_id | student_id(FK), leave_type, start_date, end_date, reason, status, apply_time, teacher_id(FK,NOT NULL), approve_time, approve_comment |

### 视图

| 视图 | 用途 |
|------|------|
| v_student_leave_detail | 学生请假详情（关联学生+班级） |
| v_pending_leave | 筛选待审批的请假申请 |
| v_class_leave_statistics | 按班级统计请假数量 |
| v_approval_history | 已审批记录历史 |

### 索引

| 索引 | 字段 | 用途 |
|------|------|------|
| idx_leave_student_id | LeaveRecord(student_id) | 学生查自己的请假记录 |
| idx_leave_status | LeaveRecord(status) | 班主任筛待审批列表 |
| idx_leave_apply_time | LeaveRecord(apply_time) | 按时间排序 |
| idx_leave_teacher_id | LeaveRecord(teacher_id) | 班主任看审批历史 |
| idx_leave_date_range | LeaveRecord(start_date, end_date) | 按日期范围筛选 |
| idx_student_class_id | Student(class_id) | 按班级查学生 |
| idx_teacher_class_id | Teacher(class_id) UNIQUE | 按班级查班主任，保证1:1 |

## API 接口规划

### 认证模块
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/auth/login | 登录（传入 id + password + role，返回用户信息） |

### 学生-请假模块
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/leave | 提交请假申请 |
| GET | /api/leave/student/{studentId} | 查询自己的请假记录 |

### 班主任-审批模块
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/leave/pending/{classId} | 查询本班待审批列表 |
| PUT | /api/leave/{leaveId}/approve | 审批通过 |
| PUT | /api/leave/{leaveId}/reject | 审批驳回 |
| GET | /api/leave/teacher/{teacherId} | 查询审批历史 |

### 教务-管理模块
| 方法 | 路径 | 说明 |
|------|------|------|
| GET/POST/PUT/DELETE | /api/students | 学生CRUD |
| GET/POST/PUT/DELETE | /api/teachers | 教师CRUD |
| GET/POST/PUT/DELETE | /api/classes | 班级CRUD |
| GET | /api/stats | 统计数据 |

## 开发顺序

1. 🔲 数据库建表 SQL + 测试数据

3. 🔲 Vue 前端登录页面
4. 🔲 学生请假（提交 + 查询）
5. 🔲 班主任审批（待审批 + 审批操作）
6. 🔲 教务管理（CRUD + 统计）
